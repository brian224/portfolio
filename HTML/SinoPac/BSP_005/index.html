常見問題

  • 支援 iOS 13.0(含)、Android 8.0(含)以上設備，如您非前述版本請更新手機系統版本。

  • APP上架後將於網站公布下載連結，敬請期待。

  • 在兩款APP併行期間，兩款APP皆可登入使用，資料也皆會是最新狀態；惟同一時間，同一個會員帳號只能登入其中一款APP，以確保您的使用安全性，謝謝。

  • 如果您已是永豐銀行網路銀行會員，可立即使用身分證字號、使用者代碼、網路密碼登入體驗。若您還不是永豐銀行網路銀行會員，歡迎線上申請或於分行臨櫃處理。

  • 您可線上進行使用者代碼、網路密碼重置，或於分行臨櫃處理。

    • SIM 卡驗證：透過 SIM 卡驗證客戶留存於本行之手機門號是否相符，並輸入簡訊動態密碼進行身分驗證。
    • 晶片金融卡驗證：透過晶片金融卡搭配讀卡機輸入金融卡密碼，接續於 APP 輸入簡訊動態密碼進行身分驗證。
    • TOGO APP 臨櫃驗證：由永豐銀行分行人員查驗客戶證件並核對身分，接續於 APP 輸入簡訊動態密碼進行身分驗證。
    • 視訊驗證：由永豐銀行客服人員透過視訊查驗客戶證件並核對身分完成身分驗證。
    • NFC 晶片金融卡（僅 Android ）：使用 NFC 晶片金融卡完成裝置綁定。

  • 請先至新永豐行動銀行 APP > 更多 > 設定 > 快速登入，完成啟用後，即可至轉帳功能，輸入非約定帳號後選擇「生物辨識」進行非約定轉帳。

  • 可查詢近兩年內的交易資料，最多一次可查詢三個月的時間區間。

    若須查詢兩年以上之交易資料，請於新永豐行動銀行 APP > 首頁 >「分析」頁籤 > 點擊「AI-Driven 智慧收支帳本」，即可查詢、匯出十年以上之交易明細。

  • 每月的帳戶餘額 ( 非最高可用金額 ) 為當月月底最後一個營業日的資料。

  • 可以，您可至憑證管理申請軟體憑證之後，以該憑證進行驗證以執行大額買賣匯交割。

    • 未申請軟體憑證：單筆最低等值新臺幣 100/ 最高小於等值新臺幣 50 萬。
    • 已申請軟體憑證：單筆最低等值新臺幣 100/ 最高小於等值新臺幣 200 萬。

  • 請登入新永豐行動銀行 APP，完成行動裝置綁定，於首頁 > 右上角鈴鐺 icon > 我的通知中心 > 右上角設定 icon > APP 頁籤，即可於此頁面關閉 / 開啟推播通知服務。

    • 若您已設定「理財密碼」，則可自行於新永豐行動銀行 APP 金融卡設定功能 ( 更多 ( 選單 ) -> 右上角設定 -> 金融卡管理 ) 中，點擊啟用國外提款來開啟國外 ATM 提款功能。
    • 提醒您，啟用時需進行簡訊 OTP 驗證，故建議您可於前往海外前，先行啟用該項功能，避免海外無法收取簡訊的情況，謝謝。

  • 有的，轉帳限額會依您的身分證統一編號歸戶累計，說明如下：

    • 一般存款帳戶(臨櫃開戶)、第一類數位存款帳戶(臨櫃或自然人憑證等驗證)及第二類數位存款帳戶(永豐實體帳戶驗證)：每筆轉帳金額不得超過新台幣5萬元，每日累計不得超過新臺幣10萬元，每月累計不得超過新臺幣20萬元。企業客戶如有透過寰宇金融網申請或變更新臺幣非約定轉帳交易限額，則每日及每月累計交易限額，依其申請內容約定之。
    • 第三類數位存款帳戶(II)(他行帳戶驗證)：每筆轉帳金額不得超過新台幣1萬元，每日累計不得超過新臺幣3萬元，每月累計不得超過新臺幣5萬元。

  • 基金購物車最多可以新增 20 檔基金，單次最多可以申購 5 檔基金。

    • 單次申購 1 檔基金：可選擇以「單筆申購」或「定時定額」方式扣款。
    • 單次申購 2~5 檔基金：僅限以「定時定額」方式扣款。

  • 目前有提供：台灣自來水費、臺北自來水事業處水費、台電電費、瓦斯費、中華電信費。

  • 生效日期依各事業單位作業及根據事業單位的結帳時間會有所不同，水、電、瓦斯、中華電信費約需 3~14 個工作日。

  • 無論是在新永豐行動銀行 APP 還是大咖設定的常用帳單，只要是新永豐行動銀行 APP 可以繳費的項目（ 水 / 電 / 瓦斯 / 電信 / 他行信用卡 / 停車費 ），都可以在新永豐行動銀行 APP 常用帳單中顯示，新永豐行動銀行 APP 的常用帳單會根據「帳單提醒」的設定，於設定的時間提醒繳費，但不會跟大咖一樣自動代查是否有新的帳單要繳費。

  • 手續費依各繳費項目不同，在繳費時會請您確認繳費資訊及手續費金額，若免手續費則無須理會。
